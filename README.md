# Basic-C-Calculator
A basic C program Calculator using do while loop, switch and menu driven approach.
It allows the user to perform multiple operations repeatedly until they choose to exit.
# Features
✔ Choose new numbers anytime
✔ Addition
✔ Subtraction
✔ Multiplication
✔ Division (with zero-error check)
✔ Power (a^b)
✔ Square root of both numbers
✔ Clean menu-driven interface
✔ Uses do-while loop for continuous operation
# How to Compile and Run
Compile:
gcc calculator.c -o calculator -lm
Run:
./calculator
# Program flow
1. Program Flow
2. User enters two numbers
3. A menu is shown
4. User chooses an operation
5. The result is printed
6. The menu repeats using do-while
7. User can exit by choosing option 8
